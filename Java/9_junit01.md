## 1. Junit 이란 무엇인가??

- <code><strong>자바 프로그래밍 언어용 유닛 테스트 프레임워크이다.</strong></code>

  - 보이지 않고 숨겨진 단위 테스트를 끌어내어 정형화 시킨 뒤 단위 테스트를 쉽게 해주는 테스트 용 프레임워크이다.
  
- 외부 테스트 프로그램을 작성하여 System.out 으로 번거롭게 디버깅을 하지 않아도 됩니다.
